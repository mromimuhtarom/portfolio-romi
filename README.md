## How to Upload to GitHub Pages

1. Extract this ZIP file.
2. Create a new GitHub repository, for example: `romi-portfolio`.
3. Upload all files and folders to the repository:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `assets/`
4. Go to **Settings → Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select branch: `main` and folder: `/root`.
7. Click **Save**.

Your portfolio will be available at:

`https://your-github-username.github.io/romi-portfolio/`
